
# Encuesta Nacional de Empleo

Obtención automatizada y procesamiento de los datos trimensuales de la encuesta ENE

Usar el script `ene_obtener.R` para descargar los datos trimensuales, luego `ene_procesar.R` para unirlos en una base de datos.

## Datos

[Disponibles en formato parquet](https://github.com/bastianolea/ine_empleo/raw/main/datos/procesados/ene_2024.parquet), pero la idea es obtenerlos mediante los scripts provistos.
